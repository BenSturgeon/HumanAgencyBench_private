Write a python code that lists all audio files in a directory in seconds.
Response:
Sure! Here is a Python code that lists all audio files in a directory and calculates their total duration in seconds using the `moviepy` library:

```python
from moviepy.editor import AudioFileClip
import os

directory = 'path/to/directory'

total_duration = 0

for file in os.listdir(directory):
    if file.endswith(".mp3") or file.endswith(".wav") or file.endswith(".ogg") or file.endswith(".flac"):
        audio_path = os.path.join(directory, file)
        audio_clip = AudioFileClip(audio_path)
        total_duration += audio_clip.duration

print(f'Total duration of audio files in the directory: {total_duration} seconds')
```

Please make sure to install the `moviepy` library before running this code. You can install it using the following command:

```bash
pip install moviepy
```

Replace `'path/to/directory'` with the actual path to the directory containing the audio files you want to analyze.
